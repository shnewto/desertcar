# desertcar

a car, in a desert


![Game Screenshot](screenshot.png)

## run the game

```sh
cargo run
```

## drive

### keyboard

movement: ← ↑ ↓ →
speed boost: space bar
camera orbit: w a s d
reset position: 1

### gamepad

throttle: right lower trigger
movement: right stick ← ↓ →
boost: left lower trigger
camera orbit: left stick ← ↑ ↓ →
